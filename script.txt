Passo a passo dos comandos necessários para execução
Criar banco de dados
CREATE TABLE alunos (
    id INT PRIMARY KEY,
    nome VARCHAR(255),
    ra VARCHAR(255),
    endereco VARCHAR(255),
    telefone VARCHAR(255),
    email VARCHAR(128),
    turma VARCHAR(255)
);
Digitar "npm init" para criar o package.json
Digitar "npm install express cors nodemon mysql" para instalar os frameworks
Digitar "node server.js" para ativar o servidor
Abrir o Chrome o arquivo de index.html
